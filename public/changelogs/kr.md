# 주요 업데이트 - Kikitan 번역기 v1.3.0
이번 업데이트는 Kikitan에 많은 새로운 기능을 제공합니다!

**새로운 기능:**

*   **Google Gemini 통합:**
    *   Google Gemini를 사용한 번역 지원이 추가되었습니다. 이전의 WebSpeech와 Google 번역 조합보다 더 나은 성능을 제공할 것으로 기대됩니다. 아직 실험적이지만 시도해 볼 가치가 있습니다.
    *   Gemini는 음성을 텍스트로 변환하고 번역하여 더 자연스러운 번역을 제공합니다.
*   **텍스트 기반 번역 입력:**
    *   중지 버튼 옆에 있는 버튼을 사용하여 말하는 대신 텍스트 상자에 입력하여 번역할 수 있습니다.
*   **향상된 빠른 시작 메뉴:**
    *   빠른 시작 메뉴에 필수 설정이 포함되어 사용자가 Kikitan을 처음 시작할 때 기본 설정을 지정할 수 있습니다. 이를 통해 최초 실행 경험이 개선되고 쉽게 시작할 수 있습니다!
*   **공지 시스템:**
    *   중요한 업데이트, 버그 수정 및 기타 관련 정보를 사용자에게 직접 전달하는 공지 시스템을 구현했습니다.
    *   외부 링크 버튼 옆에 있는 새 버튼을 통해 Discord 서버에 참여할 수도 있습니다!

**개선 사항:**

*   일부 UI 불일치 및 사소한 버그가 수정되었습니다.

**버그 수정:**

**알려진 문제:**

*   WebSpeech(기본 인식기)에서 감지 문제가 발생하는 경우 Microsoft Edge를 업데이트해 보세요. 업데이트해도 문제가 해결되지 않으면 Microsoft 웹사이트에서 다시 설치하세요.

**향후 계획:**

*   다음은 데스크톱 오디오를 번역하는 것이므로 모든 사람이 Kikitan을 설치할 필요가 없습니다.  이를 위해서는 Gemini를 사용해야 할 가능성이 높지만 상황을 지켜볼 것입니다.
