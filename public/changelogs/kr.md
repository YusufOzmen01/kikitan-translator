# 소규모 업데이트 - Kikitan Translator v1.2.3

무려 1년 만에 Kikitan에 새로운 업데이트가 추가되었습니다!
원래 이번 업데이트에는 Google Gemini 기반의 새로운 인식기와 데스크톱 오디오 번역 기능이 포함될 예정이었지만, Google이 12월 15일에 무료 Live API를 종료하면서 해당 기능은 완전히 사용할 수 없게 되었습니다. 이 기능을 사용하려면 API 비용을 지불해야 하는데, 저는 사용자에게 어떤 비용도 요구하고 싶지 않았습니다. 그래서 안타깝게도 더 나은 대안을 찾을 때까지 해당 업데이트를 보류해야 합니다.

**새로운 기능:**
* **텍스트 기반 번역 입력:**
  * 중지 버튼 옆의 버튼을 사용하여 이제 음성 대신 텍스트 박스에 입력해 번역할 수 있습니다.
* **향상된 빠른 시작 메뉴:**
  * 빠른 시작 메뉴에 필수 설정이 추가되어 Kikitan을 처음 실행할 때 기본 설정을 사용자 지정할 수 있습니다. 이를 통해 첫 실행 경험이 개선되고 더 쉽게 시작할 수 있습니다!
* **텍스트 기반 번역 입력:**
  * 중지 버튼 옆의 버튼을 사용하여 이제 음성 대신 텍스트 박스에 입력해 번역할 수 있습니다.
* **메시지 기록:**
  * 메인 페이지에 편리하게 배치된 버튼을 통해 이제 메시지 기록을 확인할 수 있습니다!

**개선 사항:**
* 일부 UI 불일치 문제 및 작은 버그가 수정되었습니다
* 마이크가 간헐적으로 인식되지 않던 문제가 해결되었습니다!

**알려진 문제:**
* 음성이 인식되지 않는 문제가 발생한다면 Microsoft Edge를 업데이트해 보세요. 업데이트로 해결되지 않으면 Microsoft 웹사이트에서 재설치하세요. 그래도 해결되지 않으면 %localappdata%\com.github.yusufozmen01.kikitantranslator 폴더를 삭제한 후 Kikitan을 다시 실행하세요.

**버그를 제보하고 싶다면 메인 페이지의 버튼을 통해 Discord에 꼭 참여해주세요!!**

**향후 계획:**
* Gemini를 대체할 적절한 솔루션을 찾아 지난 1년간 작업해 온 업데이트를 출시할 수 있기를 바랍니다. 원래 이 업데이트는 훨씬 전에 출시되었어야 했지만, 제가 좋아하는 것들과 Kikitan에 집중하지 못하게 만드는 많은 일들이 있었고, 그로 인해 업데이트가 크게 지연되었습니다. GitHub를 확인하면 해당 풀 리퀘스트가 1년 이상 열려 있는 것을 볼 수 있습니다. 업데이트가 늦어진 점에 대해 사과드리며, Google이 무료 Live API를 제거한 것에 진심으로 분노하고 있습니다. 정말로 말하자면, Google 정말 싫습니다.
