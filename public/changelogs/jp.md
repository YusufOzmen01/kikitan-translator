# マイナーアップデート - Kikitan Translator v1.2.3

約1年ぶりに、Kikitanに新しいアップデートが追加されました！
本来このアップデートでは、Google Gemini ベースの新しい認識エンジンとデスクトップ音声翻訳機能を搭載する予定でしたが、Google が 12 月 15 日に無料の Live API を終了してしまい、この機能は完全に使えなくなりました。この機能を実装するには API の利用料金が必要となり、私はユーザーに料金を支払わせたくありません。そのため、より良い代替手段を見つけるまで、このアップデートは保留とさせていただきます。

**新機能：**
* **テキスト入力による翻訳：**
  * 停止ボタンの横にあるボタンを使って、音声の代わりにテキストボックスへ入力して翻訳できるようになりました。
* **強化されたクイックスタートメニュー：**
  * クイックスタートメニューに重要な設定項目が追加され、Kikitan の初回起動時に基本設定をカスタマイズできるようになりました。これにより初回使用時の体験が向上し、より簡単に使い始められます！
* **テキスト入力による翻訳：**
  * 停止ボタンの横にあるボタンを使って、音声の代わりにテキストボックスへ入力して翻訳できるようになりました。
* **メッセージ履歴：**
  * メインページに便利に配置されたボタンから、メッセージ履歴を確認できるようになりました！

**改善点：**
* 一部の UI の不整合や小さなバグを修正しました
* マイクが時々認識されない問題を修正しました！

**既知の問題：**
* 音声が検出されない場合は、Microsoft Edge を更新してみてください。更新しても解決しない場合は、Microsoft の公式サイトから再インストールしてください。それでも解決しない場合は、%localappdata%\com.github.yusufozmen01.kikitantranslator フォルダを削除してから Kikitan を再起動してください。

**バグ報告を行いたい場合は、メインページのボタンから Discord に参加してください！！**

**今後の予定：**
* Gemini の代替となる適切なソリューションを見つけ、過去1年間取り組んできたアップデートをリリースできることを願っています。本来であれば、このアップデートはずっと前に公開されているはずでしたが、私自身の生活の中で、好きなことや Kikitan に集中できない出来事が多くあり、大幅な遅延となってしまいました。GitHub を確認すると、プルリクエストが1年以上前から開いたままになっているのが分かると思います。更新が遅れてしまったことをお詫びしますし、無料の Live API を削除した Google には本当に腹が立っています。本当に、Google のことが嫌いです。
