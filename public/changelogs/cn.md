# 小型更新 - Kikitan 翻译器 v1.2.3

在整整一年之后，Kikitan 终于迎来了新的更新！
原本这次更新计划加入基于 Google Gemini 的新识别器以及桌面音频翻译支持，但 Google 于 12 月 15 日关闭了免费的 Live API，这使得该功能完全无法使用，因为这将要求用户为 API 付费，而我并不希望向用户收取任何费用。因此，我不得不遗憾地将该更新搁置，直到我找到更好的替代方案。

**新功能：**
* **基于文本的翻译输入：**
  * 现在，你可以通过停止按钮旁边的按钮，在文本框中输入内容进行翻译，而不必使用语音。
* **增强的快速开始菜单：**
  * 快速开始菜单现在包含了一些核心设置，允许用户在首次启动 Kikitan 时自定义基本设置。这改善了首次使用体验，让上手更加容易！
* **基于文本的翻译输入：**
  * 现在，你可以通过停止按钮旁边的按钮，在文本框中输入内容进行翻译，而不必使用语音。
* **消息历史记录：**
  * 你现在可以通过主页上一个方便的位置按钮查看你的消息历史！

**改进内容：**
* 修复了一些 UI 不一致问题和小错误
* 修复了麦克风有时无法被检测到的问题！

**已知问题：**
* 如果你遇到语音无法被检测的问题，请尝试更新 Microsoft Edge。如果更新无法解决问题，请从微软官网重新安装。如果仍然无效，请删除 %localappdata%\com.github.yusufozmen01.kikitantranslator 文件夹并重新启动 Kikitan。

**如果你想提交错误报告，请务必通过主页上的按钮加入 Discord！！**

**未来计划：**
* 我希望能找到一个合适的 Gemini 替代方案，从而发布我过去一年一直在开发的更新。原本这个更新应该早就发布了，但由于我生活中发生了很多事情，导致我对自己喜欢的事情和 Kikitan 都有些分心，因此更新被严重拖延。如果你查看 GitHub，会发现该拉取请求已经开放一年多了。对于迟迟未能发布新更新，我深表歉意，同时我也真的很讨厌 Google 移除他们的免费 Live API。说真的，我讨厌你，Google。
